# Astra AI – Landing Page (Next.js)

Welcome to the **Astra AI Landing Page**, a modern and responsive marketing site built with **Next.js** and **Tailwind CSS**. This project highlights the core features and offerings of Astra AI, a no-code AI-powered website builder.

> ✨ Build your next idea and ship your dream site – zero code, maximum speed.

## 🛠️ Tech Stack

* **Framework**: [Next.js](https://nextjs.org/)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Icons**: [Lucide-react](https://lucide.dev/)
* **Hosting**: Vercel / Netlify / Render (recommended)

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/sanjayattelli29/Astra-ai-landing-page-design.git
cd astra-landing-page
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

Open `http://localhost:3000` to see the site.

---

## 🏢 Deployment

Build and deploy your project using:

```bash
npm run build
npm start
```

Recommended Platforms:

* [Vercel](https://vercel.com)
* [Netlify](https://netlify.com)
* [Render](https://render.com)

## ⚙️ Customization

* Update site branding in `index.tsx` and `Navbar.tsx`
* Replace the banner image in `/public/banner.png`
* Edit pricing plans in `components/PricingSection.tsx`
* Add real testimonials or integrate with Twitter API
* Configure newsletter with Mailchimp or ConvertKit
